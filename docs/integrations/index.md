# Integrations

Xberg integrates with AI frameworks, databases, and search engines — bringing document extraction into your existing stack. Each integration is a standalone package published on PyPI.

---

## Available integrations

| Integration | Framework                                         | Package                                                                                                                          | Docs                                                                                                |
| ----------- | ------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |

| LangChain   | [LangChain](https://www.langchain.com/)           | [`langchain-xberg`](https://pypi.org/project/langchain-xberg/)                                                           | [GitHub](https://github.com/xberg-io/langchain-xberg)                                      |
| LlamaIndex  | [LlamaIndex](https://www.llamaindex.ai/)          | [`llama-index-readers-xberg`](https://pypi.org/project/llama-index-readers-xberg/)                                       | [GitHub](https://github.com/xberg-io/llama-index-xberg)                                    |
